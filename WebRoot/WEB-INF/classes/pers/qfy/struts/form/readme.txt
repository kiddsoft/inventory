这些都是 表单(form)

作用是：
当JSP需要将一些信息提交到action中时，比如说添加商品这个功能，则步骤是：

1，在JSP中填写了内容，点击了提交
2，提交了之后，内容会保存到指定的form中，(具体看strut-config.xml中的名字)。这时候数据会提交到服务器
3，进入服务器中的action之后，可以获取到之前填写的form，也就是可以获取到用户填写的数据了
4，然后对这些数据进行处理，进行保存(根据功能逻辑，或者查询，或者修改，或者删除等）
5，操作完了之后，action又使用return mapping.findForward("XXX");来到达指定位置
6，如果XXX是一个action，则是又跳入到action
7，如果XXX是一个jsp，则用户界面就会显示这个JSP页面
