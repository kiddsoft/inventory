用户表
tb_user
username	char	用户ID
password	char	密码
name		char	姓名
level		int	等级，是否需要再建一个子表显示这个。0表示等级最高，1为普通管理员，2为普通用户
superior	char	上级的用户ID
isstock		int	是否允许进货
issell		int	是否允许出货
ismgr		int	是否允许管理订单
createtime	time	创建时间




商品表
tb_commodity
cno		char	商品号
name		char	商品名称
inprice		int	建议进货价格
outprice	int	建议出货价格
count		int	库存数量
description	char	说明




订单表
tb_indent
ino		char	订单ID
cno		char	商品编号
username	char	订单的申请人用户ID
superior	char	审核人的用户ID
icount		int	数量
price		int	销售价格
isoutsell	int	是否出货订单。0表示进货，1表示为出货
istate		int	审核状态。0表示待审核，1表示成功，2表示审核失败  -- 其中1与2表示审核结束了
itime		time	订单时间
endtime		time	审核时间
